# Support

Thanks for using Brooks AI HUB! This page lists support channels and the best place to file issues.

## Contact options

- **Email**: [support@brooksaihub.app](mailto:support@brooksaihub.app)
- **Discord**: Join the community at https://discord.gg/brooks-aihub
- **GitHub Discussions**: https://github.com/NorthernAmericana/brooks-aihub-chat-175/discussions

## Where to file issues

- **Bug reports & feature requests**: Use [GitHub Issues](https://github.com/NorthernAmericana/brooks-aihub-chat-175/issues) for actionable tasks, regressions, or tracked work.
- **Questions & ideas**: Start a topic in [GitHub Discussions](https://github.com/NorthernAmericana/brooks-aihub-chat-175/discussions) for help, feedback, or product ideas before filing an issue.

## What to include in bug reports

- Steps to reproduce
- Expected vs. actual behavior
- Screenshots or recordings (if applicable)
- Environment details (browser, device, OS)
- Relevant logs or error messages
